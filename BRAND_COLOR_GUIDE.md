# Giftwala Brand Color Guide

## Primary Brand Colors

### Primary Palette
- **Primary Brand Color**: `#DC2626` (Warm Red) - Main brand identity
- **Primary Dark**: `#B91C1C` (Deep Red) - Hover states, emphasis
- **Primary Light**: `#FEE2E2` (Light Red) - Backgrounds, subtle accents

### Secondary Palette
- **Secondary Brand Color**: `#F59E0B` (Amber/Gold) - Premium accents, highlights
- **Secondary Dark**: `#D97706` (Dark Amber) - Hover states
- **Secondary Light**: `#FEF3C7` (Light Amber) - Backgrounds

### Accent Colors
- **Accent Pink**: `#EC4899` (Vibrant Pink) - CTAs, special highlights
- **Accent Purple**: `#A855F7` (Purple) - Gradients, variety
- **Accent Rose**: `#F43F5E` (Rose) - Warm accents

## Usage Guidelines

### Primary Colors
- Use **Primary Red** for main CTAs, buttons, and key brand elements
- Use **Primary Dark** for hover states and emphasis
- Use **Primary Light** for backgrounds and subtle highlights

### Secondary Colors
- Use **Amber/Gold** for premium features, special offers, and luxury elements
- Use **Amber Dark** for hover states on gold elements
- Use **Amber Light** for backgrounds and subtle gold accents

### Accent Colors
- Use **Accent Pink** for secondary CTAs and special promotions
- Use **Accent Purple** in gradients for variety and visual interest
- Use **Accent Rose** for warm, inviting accents

## Color Combinations

### Primary Gradient
```
from-red-600 via-red-500 to-rose-500
```

### Premium Gradient
```
from-amber-500 via-yellow-500 to-orange-500
```

### Accent Gradient
```
from-pink-500 via-rose-500 to-red-500
```

## Neutral Colors

- **Text Primary**: `#1F2937` (Gray-800)
- **Text Secondary**: `#6B7280` (Gray-500)
- **Background**: `#FFFFFF` (White)
- **Background Light**: `#F9FAFB` (Gray-50)
- **Border**: `#E5E7EB` (Gray-200)

## Implementation

All colors are implemented using Tailwind CSS classes. The brand colors are used consistently across:
- Buttons and CTAs
- Navigation elements
- Product cards and badges
- Footer and header
- Form elements
- Hover states and interactions

---

**Note**: This color guide is based on typical gift/e-commerce branding. Colors can be adjusted based on the actual logo analysis.
